## Quick Reference

- Always end with: no text no watermark
- Always add: Square composition, no faces
- Generate 4 per prompt, pick best match to app's warm palette
- If too cool/blue, add "warm amber tones"
- Avoid: realistic, detailed, beautiful, epic
- Use: painterly, atmospheric, cinematic, intimate
